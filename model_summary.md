# Churn Prediction Model Summary

## Model Used
Random Forest Classifier

## Features Used
- Total Orders
- Total Spend
- Average Rating

## Evaluation
The model was trained using an 80-20 train-test split.

Metrics evaluated:
- Accuracy
- Precision
- Recall
- F1-Score

## Key Findings

- Customers with lower spending show higher churn risk.
- Customers with fewer orders are more likely to churn.
- Average customer rating contributes to churn prediction.
